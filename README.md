# wintersmith-gherkin

[Wintersmith](https://github.com/jnordberg/wintersmith) gherkin plugin

## Installing

Install globally or locally using npm

```
npm install [-g] wintersmith-gherkin
```

and add `wintersmith-gherkin` to your config.json

```json
{
	"plugins": [
		"wintersmith-gherkin"
	]
}
```

## Running tests

```
npm install
npm test
```
